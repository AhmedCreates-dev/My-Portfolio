# Personal Portfolio Website

A modern and responsive personal portfolio website built using **HTML5** and **CSS3**. This project showcases my skills, projects, and professional profile through a clean and user-friendly interface.

## Features

* Responsive design for desktop and mobile devices
* Clean and modern UI
* Smooth navigation
* Projects showcase section
* Skills and about section
* Contact information section

## Technologies Used

* HTML5
* CSS3

## Purpose

This portfolio was developed to strengthen my front-end development skills and create a professional online presence to showcase my work and achievements.
